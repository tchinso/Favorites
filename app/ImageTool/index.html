도구 A-1 - 이미지 변환 (ImageConverter 통합)

ffmpeg 없이 동작합니다. 드래그앤드롭, 다중 파일 변환, Animated → 정지 변환 시 프레임 선택 팝업을 지원합니다.

파일을 드래그해서 놓으세요.

또는 파일 선택 버튼으로 여러 개를 추가할 수 있습니다.
